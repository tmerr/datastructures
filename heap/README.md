Heap
====

A by-the-book heap for sorting or use as a priority queue, that is made generic through C++ templating.

| Operation     | Time        |
| ------------- |:-----------:|
| heapify       | O(n)        |
| heapSort      | O(n lg n)   |
| insert        | O(lg n)     |
| peek          | O(1)        |
| pop           | O(lg n)     |
